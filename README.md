# Email-SImulator-Workshop
Email Simulator 📧
A simple email system simulator built in Python. Create users, send emails, manage your inbox, mark emails as read, view full messages, and delete them.
Features:

Object-Oriented design with User, Inbox, and Email classes
Automatic timestamp
Read/Unread status
Clean terminal interface

Perfect project for learning Object-Oriented Programming (OOP) in Python.
